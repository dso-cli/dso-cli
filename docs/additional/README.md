# Additional Documentation

This directory contains additional documentation files that are not part of the main VitePress documentation site.

## Files

- **BUILD.md** - Build instructions and compilation details
- **CHECKLIST.md** - Development checklist
- **CONTRIBUTING.md** - Contribution guidelines
- **FEATURES.md** - Feature list and descriptions
- **INSTALL.md** - Detailed installation guide
- **OLLAMA_INTEGRATION.md** - Ollama integration details
- **PLATFORMS.md** - Platform-specific installation instructions
- **PROJECT_STRUCTURE.md** - Project structure and architecture
- **QUICK_BUILD.md** - Quick build instructions
- **QUICK_START.md** - Quick start guide
- **ROADMAP.md** - Project roadmap and future plans

## Main Documentation

For the main documentation site with VitePress and Catppuccin theme, see the parent `docs/` directory.

To view the documentation site:

```bash
cd docs
npm install
npm run dev
```

Access at `http://localhost:5173`
